## Tentang Book Ini

Jupyter Book ini berisi berbagai pembahasan mengenai proses penambangan data mulai dari pemahaman data, persiapan data, pengolahan data, hingga analisis menggunakan metode tertentu seperti Naive Bayes.

Seluruh isi dalam book ini disusun sebagai bentuk dokumentasi pembelajaran sekaligus pengembangan pemahaman terhadap konsep Data Mining yang diterapkan secara langsung menggunakan tools dan dataset yang relevan.
